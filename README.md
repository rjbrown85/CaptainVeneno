# Captain Veneno Website

Heavy, hard alt-rock brewed in feedback, dread, and bad decisions.

## How to Host this Website on GitHub Pages

To make this website live and visible to anyone on the web:

1. **Create a GitHub Repository**:
   - Go to [GitHub](https://github.com/) and create a new repository (e.g., `captain-veneno`).

2. **Upload the Files (Two Options)**:

   * **Option A (Recommended - Root Site)**:
     - Open this `Captain Veneno website` folder on your computer.
     - Select **all the files inside it** (like `index.html`, `connect.html`, `.nojekyll`, and all the `.png` images).
     - Drag and drop them directly into your GitHub repository page.
     - Your live URL will be: `https://<your-username>.github.io/captain-veneno/`

   * **Option B (Uploading the Whole Folder)**:
     - Drag the **entire `Captain Veneno website` folder** into your GitHub repository page.
     - Your live URL will be: `https://<your-username>.github.io/captain-veneno/Captain%20Veneno%20website/`
     - *(Note: All links and images are fully relative, so the site will work perfectly under this subfolder path as well!)*

3. **Enable GitHub Pages**:
   - In your repository, click the **Settings** tab at the top.
   - Click on **Pages** in the left sidebar menu (under the "Code and automation" section).
   - Under **Build and deployment**, set **Source** to `Deploy from a branch`.
   - Set the **Branch** dropdown to `main` (or `master`) and the folder path to `/ (root)`.
   - Click **Save**.

4. **View the Website**:
   - Wait 1 to 2 minutes for GitHub to build the site, and then open your live URL!
